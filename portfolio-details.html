Project Detail

● LapSell Corner is second hand laptop buy and sell related MERN stack website.

● This website is fully responsive for mobile, tablet, and desktop devices.

● It has a separate dashboard for buyer, seller, and admin

● Buyer can book any product and also can report. And admin can remove the reported product.

● If the user isn't login he/she won't be able to order anything, they will be redirected to the login page.

● Technology Used: HTML | CSS | Tailwind | React.js | Node.js | Express.js | Mongodb | Firebase | React Route | React hook form | daisyUI
